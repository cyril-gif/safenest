# 🛡️ SafeNest — Family Location Sharing

Real-time location sharing between father and son using Firebase.

## Setup

```bash
npm install
npm run dev
```

## Deploy to Netlify / Vercel

```bash
npm run build
```
Then upload the `dist/` folder or connect your GitHub repo.

## How it works

1. Father registers with a **Family Code**
2. Son registers with the **same Family Code** on his phone
3. Son taps the button to go live
4. Father sees son's location on the radar map in real time
